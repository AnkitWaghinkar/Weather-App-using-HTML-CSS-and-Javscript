# Weather App

## Description
A weather app built with HTML, CSS, and JavaScript. It fetches and displays the current weather data based on user location or input city.

## Features
- Display current weather, temperature, and other weather details.
- Fetch data using an external weather API (OpenWeatherMap).
- Search for weather by city name.

## Technologies Used
- HTML  
- CSS  
- JavaScript (API Integration).
